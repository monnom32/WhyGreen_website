# 🌬️ Wind Clicker — Инструкция по установке

## Структура папки

```
wind-clicker/
├── index.html      ← страница игры (не трогай)
├── style.css       ← стили (не трогай)
├── game.js         ← логика (не трогай)
├── config.js       ← ВСЕ НАСТРОЙКИ ЗДЕСЬ ✅
├── README.md       ← эта инструкция
└── images/
    ├── bg.png           ← фон (любой размер, например 1920×1080)
    ├── blade.png        ← пропеллер (рекомендуется 200×200px, PNG с прозрачным фоном)
    ├── tower.png        ← башня ветряка (рекомендуется 80×280px, PNG с прозрачным фоном)
    ├── upg_blades.png   ← иконка улучшения "Длинные лопасти" (64×64px)
    ├── upg_gear.png     ← иконка улучшения "Редуктор" (64×64px)
    ├── upg_auto.png     ← иконка улучшения "Авторотация" (64×64px)
    ├── upg_farm.png     ← иконка улучшения "Ветроферма" (64×64px)
    └── upg_super.png    ← иконка улучшения "Суперклик" (64×64px)
```

> ⚠️ Если картинки не найдены — игра показывает emoji-заглушки и работает нормально.
> Добавляй картинки постепенно, по одной.

---

## Как добавить картинки

1. Создай папку `images/` рядом с `index.html`
2. Положи туда файлы с нужными именами (см. структуру выше)
3. Готово — картинки подхватятся автоматически

### Советы по картинкам

| Файл | Рекомендуемый размер | Формат | Совет |
|------|---------------------|--------|-------|
| `bg.png` | 1920×1080 или больше | PNG/JPG | Пейзаж с небом, поле |
| `blade.png` | 200×200px | **PNG с прозрачным фоном** | Три лопасти, центр — центр картинки |
| `tower.png` | 80×280px | **PNG с прозрачным фоном** | Только башня без лопастей |
| `upg_*.png` | 64×64px | PNG | Иконки улучшений |

## Как настроить игру (config.js)

Открой `config.js` — там всё подписано. Основные параметры:

```js
baseClickEnergy: 1,   // сколько ⚡ даёт один клик
spinDuration: 1200,   // мс — как долго крутится пропеллер
```

### Добавить новое улучшение

Скопируй блок внутри массива `upgrades: [...]` и измени:

```js
{
  id: "my_upgrade",           // уникальный id (латиница)
  name: "Моё улучшение",
  description: "+5 ⚡ за клик",
  cost: 50,                   // стоимость при первой покупке
  icon: "images/my_icon.png", // путь к иконке
  emoji: "🔥",                // запасной символ
  effect: { type: "click", value: 5 },   // тип: "click" или "passive"
  maxBuys: 3,                 // 0 = без ограничения
  costGrowth: 1.5             // во сколько раз дорожает
}
```

### Добавить новый факт

В массиве `facts: [...]`:
```js
{
  threshold: 300,   // при скольких накопленных ⚡ показать
  text: "Твой факт здесь!"
}
```

## Публикация на GitHub Pages

1. Создай репозиторий на GitHub (например `my-wind-game`)
2. Загрузи всю папку `wind-clicker/` в корень репозитория
3. Зайди в **Settings → Pages → Source: Deploy from branch → main / root**
4. Через 1–2 минуты игра будет доступна по адресу:
   `https://ТВО_ИМЯUSER.github.io/my-wind-game/`

### Ссылка с WordPress

В редакторе WordPress добавь HTML-блок:
```html
<a href="https://ТВОЙ_ЮЗЕР.github.io/my-wind-game/" target="_blank">
  🎮 Играть в Wind Clicker
</a>
```
Или встрой через iframe (если тема разрешает):
```html
<iframe src="https://ТВОЙ_ЮЗЕР.github.io/my-wind-game/"
        width="100%" height="700" frameborder="0"
        style="border-radius:12px; box-shadow: 0 4px 20px rgba(0,0,0,0.1)">
</iframe>
```
